# SCENE_ENGINE_SYSTEM — generator scen jako system

Cel: generować FRAGMENT RZECZYWISTOŚCI, nie "opis zdjęcia".
Każda scena = 5 obowiązkowych warstw.

```
Złota reguła:
If it looks intentional → it's fake
If it looks accidental → it's real
```

---

## STRUKTURA SCENY (zawsze 5 warstw)

```
[SUBJECT_STATE]    — co się dzieje z człowiekiem (fizycznie)
[ENVIRONMENT]      — gdzie jest (brud, chaos, kontekst)
[PHYSICS]          — ciężar, kieszenie, deformacje
[LIGHT]            — jedno złe źródło
[INTENT]           — dlaczego zdjęcie powstało przypadkiem
```

---

## TEMPLATE DO WYPEŁNIENIA

```
fragment of a day, not a posed photo,

[SUBJECT_STATE]
[ENVIRONMENT]
[PHYSICS]
[LIGHT]
[INTENT]
```

## GENERATOR — LOSUJ Z KAŻDEJ KATEGORII

### SUBJECT_STATE (losuj 1-2)
- zmęczony po pracy
- czeka za długo
- nie wie co robi dalej
- coś go boli (kolano / plecy)
- rozkojarzony
- lekko zirytowany
- "wyłączony"
- lekko pod wpływem (matowość, dullness)

### ENVIRONMENT (losuj 1)
- kuchnia z bałaganem
- przystanek z zarysowaną ławką
- klatka schodowa (brudne ściany)
- parking (mokry asfalt)
- łazienka (zacieki na lustrze)
- sklep nocny (lodówka z napojami)
- pokój po nocy (gnieciona pościel)
- balkon z widokiem na bloki

### PHYSICS (obowiązkowe — minimum 2)
- telefon wypycha prawą kieszeń
- papierosy deformują lewą kieszeń
- jedna noga bardziej obciążona
- koszulka przekręcona przy kołnierzu
- rękaw podwinięty tylko z jednej strony
- torba/pasek podnosi jedno ramię
- klucze robią asymetryczny garb

### LIGHT (zawsze złe — losuj 1)
- górne fluorescencyjne (cienie pod oczami)
- latarnia uliczna (żółta / sodowa)
- światło z lodówki (zimne, bije w twarz)
- ekran telefonu / monitora (zielonkawy tint)
- odbicie od auta (żółtozielony policzek)
- mieszane WB (okno + lampa)

### INTENT (losuj 1)
- zdjęcie zrobione przez przypadkowego przechodnia
- ktoś trzymał telefon krzywo
- odbicie w lustrze / szybie
- kadr ucięty za wcześnie
- brak kontaktu z obiektywem
- zdjęcie powstało bo telefon był już w ręce
