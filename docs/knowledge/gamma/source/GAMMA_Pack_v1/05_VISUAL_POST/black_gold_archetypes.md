Zasada: GOLD = fizyczny metal, nie magia. BLACK = głęboki cień, kurz, szum cyfrowy.
Nie "ładne grafiki" — fizyczne dowody.

```
Najsilniejszy efekt:
GOLD = jedyny element "nie z tego świata"
BLACK + POSTAĆ = boleśnie prawdziwe
```

## GAMMA_POST_INSTRUCTION (dla wszystkich archetypów)

```
Apply 35mm film grain.
Introduce slight chromatic aberration on edges.
Crush blacks but leave digital noise in 5% luminance range.
Desaturate midtones by 15%.
Sharpen ONLY defects (scars, pores, sweat).
Avoid all upscale artifacts that smooth the image.
```
