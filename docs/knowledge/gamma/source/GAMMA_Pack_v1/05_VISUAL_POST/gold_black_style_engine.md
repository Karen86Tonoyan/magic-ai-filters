## KOLORY (konkret)

### BLACK (tło):
```css
#0b0b0b   /* primary background */
#111111   /* secondary */
#1a1a1a   /* cards / elevated surfaces */
```
NIE używaj pure black (#000000) — za płaski.

### GOLD (3 warstwy — każda ma rolę):
```css
#c6a85a   /* base gold — główny kolor akcent */
#9f8443   /* dirty gold — cień, subtleties */
#e7d38a   /* highlight gold — rzadko, max 10% użycia */
```

Złoto NIGDY:
- nie jest płaskie
- nie jest UI gradientem
- zawsze wygląda jak MATERIAŁ (metal, nie kolor)

### TEKST:
```css
#eaeaea   /* primary text */
#8a8a8a   /* secondary / labels */
```

## LAYOUT LOGIC

- duże marginesy (70-120px desktop)
- elementy NIE idealnie wycentrowane (lekka asymetria)
- jedna rzecz na sekcję — nie upychasz
- duże powietrze = premium signal
- lekki chaos (kontrolowany) — 1-2 elementy nieznacznie przesunięte
